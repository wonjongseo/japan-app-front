# Japan Voca App

## 제작 의도

1. 본인이 일본어 단어 앱을 이용해 공부하면서 추가 되었으면 하는 기능을 넣어서 제작함.
2. Flutter 를 이용한 앱 개발 연습

## 개발 환경

### Backend

1.  NodeJs - Express
2.  Mongodb - Mongoose

### FrontEnd

1.  Flutter
